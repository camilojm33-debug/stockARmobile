# Branding Assets

This folder centralizes all visual identity files used by StockArmobile.

Current files are temporary placeholders copied from existing icons and will be replaced later with final brand assets.

## Files and usage

### logo.png
Logo principal utilizado en toda la aplicacion.

### logo-dark.png
Version para fondos claros.

### logo-light.png
Version para fondos oscuros.

### isotipo.png
Icono principal de la aplicacion.

### isotipo-dark.png
Version oscura del isotipo.

### isotipo-light.png
Version clara del isotipo.

### favicon.ico
Icono del navegador.

### apple-touch-icon.png
Icono para dispositivos Apple.

### icon-192.png
Icono PWA Android.

### icon-512.png
Icono PWA Android alta resolucion.

### splash.png
Pantalla de carga.

### logo-pdf.png
Logo utilizado en reportes y PDFs.

### logo-email.png
Logo utilizado en correos automaticos.

## Notes

- Replacement target path for future updates: `static/images/branding/`.
- Do not remove legacy assets yet; compatibility is still preserved.
